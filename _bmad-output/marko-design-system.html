Hebrew Rules

No italics for Hebrew emphasis Hebrew italics are hard to read. Use bold or color instead.
Line height 1.7 for body Hebrew characters are taller and denser — generous line height improves readability.
No letter-spacing for Hebrew Hebrew is not designed for loose tracking. Keep at 0.
Code always LTR Inline code and code blocks: direction: ltr; unicode-bidi: embed;

Rules

1 Backgrounds use HSL 155° (emerald), never 0° (gray)
2 Borders use alpha-blended emerald: rgba(110,231,183,0.12)
3 Shadows are neutral-dark (no emerald tint on dark bg)

Hebrew-first. Every layout decision accounts for right-to-left. Logical CSS properties are mandatory.

Reduced Motion

prefers-reduced-motion

All decorative animations are disabled when the user has requested reduced motion. Functional transitions (opacity for visibility) use 0.1s instead of full removal.

@media (prefers-reduced-motion: reduce) { *, *::before, *::after { animation-duration: 0.01ms !important; animation-iteration-count: 1 !important; transition-duration: 0.01ms !important; } }
